# PentoText
Pour numériser automatiquement des textes manuscripts


# Python libraries :

```console
pip3 install pytesseract
pip3 install opencv-python
pip3 install ocrmypdf
```

# External software :

```console
sudo apt install tesseract-ocr
sudo apt install libtesseract-dev
```
